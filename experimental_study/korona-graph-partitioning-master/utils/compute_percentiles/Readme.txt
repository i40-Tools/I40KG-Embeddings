Program: get_percentiles.py
===============================

About
=====

Program to obtain the percentiles values [1] of a matrix of values (e.g.  matrix of similarities)

Version
=======
0.1

License
=======

GNU GENERAL PUBLIC LICENSE Version 2.

Requirements
=============

* python3
* numpy


Usage
======

Run:
>./get_percentiles.py  <matrix_file>

where:

*  <matrix_file>: file with values in a matrix format

Running an example
==================

>./get_percentiles.py  sim_matrix.txt
